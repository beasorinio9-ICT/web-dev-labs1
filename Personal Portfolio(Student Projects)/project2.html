Pharmacy Management System

This is a Windows Forms application I built to manage medicines, inventory, and billing in a pharmacy setting. The system includes features like **login authentication**, **medicine management (add, update, delete)**, **inventory tracking**, and **XML-based data storage**.

This project helped me understand **data handling**, **search functions**, and **role-based functionality**. It gave me experience in designing user-friendly forms, creating reusable helper classes, and implementing persistent storage using XML.
