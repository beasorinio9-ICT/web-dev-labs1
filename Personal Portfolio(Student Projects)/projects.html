Space Shooter (Windows Forms)

This is a class project demonstrating inheritance, timers, collision detection, and image resource usage. Player shoots enemies while surviving waves. The project includes player health, enemy bullets, and scoring. The codebase has clear comments and resource images named consistently so Visual Studio can import them into Resources. For instructors: open the solution, ensure image resources are added, and run in Debug to test gameplay. PharmacyXMLSystem

Inventory and billing app using XML for persistent storage. Features add/update/delete/search, low stock alerts, and billing with a DataGridView interface. The XML helper class manages storage and retrieval, making the app portable and easy to inspect. The project is ideal for demonstrating practical CRUD and file handling in C#.
