Featured Projects & Details

Space Shooter (Windows Forms)

This is a class project demonstrating inheritance, timers, collision detection, and image resource usage. Player shoots enemies while surviving waves. The project includes player health, enemy bullets, and scoring. The codebase has clear comments and resource images named consistently so Visual Studio can import them into Resources. For instructors: open the solution, ensure image resources are added, and run in Debug to test gameplay. The page above contains a short README summary and the GitHub repo link (add your repo link here once uploaded).

Flappy Bird Clone (Windows Forms)

This project replicates Flappy Bird mechanics and adds shooting mechanics and scoring. Includes Start button and round-based scoring. I implemented collision detection between bullets and enemies, a restart flow, and visual assets stored in the project's resources. The project is documented so the instructor can run it locally and verify functionality during live grading.

PharmacyXMLSystem

Inventory and billing app using XML for persistent storage. Features add/update/delete/search, low stock alerts, and billing with a DataGridView interface. The XML helper class manages storage and retrieval, making the app portable and easy to inspect. The project is ideal for demonstrating practical CRUD and file handling in C#.
