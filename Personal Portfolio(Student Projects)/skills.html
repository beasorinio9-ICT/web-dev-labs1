Technical & Soft Skills

I have a foundation in HTML, CSS, and JavaScript for front-end pages, and I build desktop apps using C# and Windows Forms for school projects. I have created games (Space Shooter, Flappy clone), a Pharmacy management system using XML storage, and smaller utilities. Besides technical skills, I emphasize clean design, usability, and accessibility. I practice semantic HTML structure with proper headings and alt text for images. I use Git and GitHub for version control—making frequent, small commits to show progress during live class checkpoints. My soft skills include patience, attention to detail, willingness to ask questions, and the ability to explain my approach in simple steps during demos.
