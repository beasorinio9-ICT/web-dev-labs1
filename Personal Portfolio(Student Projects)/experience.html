Class Projects & Practical Experience

During my studies, I completed several practical projects that demonstrate applied skills in programming, UI design, and file handling. Notable projects include: a Space Shooter game to demonstrate object-oriented principles and event-driven timers; a Flappy Bird clone extended with shooting mechanics to practice collision detection and game loops; and a PharmacyXMLSystem that showcases CRUD operations and persistent storage using XML. Each project includes a README with setup instructions so instructors can run them locally. I also collaborated on small group tasks where we practiced dividing responsibilities, merging code, and presenting results during class checkpoints. Through these experiences I improved my debugging process, careful resource management for images, and writing user-friendly interfaces for peers to test.

Skills used during projects

  • Object-oriented programming (inheritance, classes)
  • Event-driven programming (timers, collisions)
  • Resource management (images, assets in Visual Studio)
  • Version control and Git workflows
