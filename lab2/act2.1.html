In this module, I learned how to:

  • Understand the basics of Git — a version control system that helps track changes in code and collaborate with others.
  • Use GitHub — a platform for hosting Git repositories, sharing projects, and collaborating with developers worldwide.
  • Create a GitHub account by visiting GitHub Sign Up, entering personal details, verifying your email, and setting up your profile.
  • Explore GitHub Pages — a feature that allows you to publish websites directly from your GitHub repository, perfect for showcasing portfolios or projects.

This module helped me connect technical tools with creative possibilities, especially in building and sharing web projects like Bean & Bea’s Corner!
